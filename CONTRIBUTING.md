# Contributing to Answer King

Thank you for your interest in contributing to the **Answer King Ecosystem**! As a part of the **SaaSSkul** framework, we value high-quality code and community feedback.

## How to Contribute
1. **Fork the Repository**: Create your own copy of the ecosystem.
2. **Create a Branch**: `git checkout -b feature/your-feature-name`.
3. **Follow Standards**: Ensure your PHP code follows WordPress Coding Standards and JS follows the `@wordpress/scripts` linting rules.
4. **Write Tests**: Add unit tests for any new keyword processing logic.
5. **Submit a Pull Request**: Provide a clear description of your changes.

## Local Development Setup
1. Clone your fork.
2. Run `composer install` and `npm install`.
3. Run `npm run start` to begin development on the React-based admin UI.
