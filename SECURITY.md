# Security Policy

## Supported Versions

We provide security updates for the following versions of Answer King:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within the **Answer King Ecosystem** or the **SaaSSkul Licensing Engine**, please do not report it publicly in the issue tracker.

Instead, please email **info@saasskul.com**.

We will acknowledge your report within 48 hours and provide a timeline for a fix. Thank you for helping keep our community safe.
