## 🏭 Framework Architecture
- **SaaSSkul Foundation Engine**: Centralized management for licensing, roles, and compliance.
- **Multi-Theme Control**: Simultaneous management of parent and child themes at the page level.

## 🚀 Local Setup Instructions

### Prerequisites
- **Local Server**: LocalWP, XAMPP, or wp-env.
- **PHP**: v8.1+
- **Composer**: For PHP library management.
- **Node.js & npm**: For building React-based admin components.

### Installation
1.  **Navigate to your WordPress plugins folder:**
    ```bash
    cd wp-content/plugins
    ```

2.  **Initialize the Ecosystem:**
    ```bash
    git clone https://github.com/FazalShahidLatif/answer-king.git
    cd answer-king
    composer install
    npm install
    npm run build
    ```

3.  **Activation:**
    - Activate the **Answer King Core** plugin.
    - Set the **SaaSSkul Core** theme as your active theme (or use a child theme).
